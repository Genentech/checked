# Creating new Check Design Objects

Instantiate a check design from a path or directory.

## Usage

``` r
new_checker(...)

new_rev_dep_checker(x, ...)
```

## Arguments

- ...:

  Additional arguments passed to `new_checker()`

- x:

  A file path, passed to
  [`plan_rev_dep_checks()`](https://Genentech.github.io/checked/reference/plan_rev_dep_checks.md)
